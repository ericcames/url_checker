# Code of Conduct

## Our Standards

This project is welcoming to all contributors. We expect participants to:

- Be respectful and constructive in all interactions
- Accept feedback graciously and offer it thoughtfully
- Focus on what is best for the project and the community
- Show empathy toward other contributors

## Scope

This code of conduct applies to all project spaces including GitHub issues, pull requests, and any other communication related to this project.

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainer at ericcames@msn.com. All reports will be handled with discretion.

## Enforcement

Project maintainers have the right to remove, edit, or reject contributions that do not align with these standards, and may take other actions they deem appropriate.
